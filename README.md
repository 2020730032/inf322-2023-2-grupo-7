# Running frontend locally

## Precondition
- nodejs 18.12
- npm 9.1.3

### Clone repository
```
git clone https://github.com/2020730032/inf322-2023-2-grupo-7
```

### Cd into frontend folder
```
cd frontend
```

### Install npm packages
```
npm install
```

### Run server
```
npm start
```
